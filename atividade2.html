1. Metadados

Definindo o título da página:

<!DOCTYPE html>
<html lang="pt-br">
<head>
 <meta charset="UTF-8">
 <meta name="description" content="Exemplo de página HTML">
 <meta name="keywords" content="HTML, exemplo, metadados">
 <title>Minha Página HTML</title>
</head>
<body>
 <!-- Conteúdo da página aqui -->
</body>
</html>

Incluindo um link para uma folha de estilo externa:

<!DOCTYPE html>
<html lang="pt-br">
<head>
 <meta charset="UTF-8">
 <link rel="stylesheet" href="estilos.css">
 <title>Minha Página com Estilo</title>
</head>
<body>
 <!-- Conteúdo da página aqui -->
</body>
</html>

2. Separação de Conteúdo

Uso de cabeçalhos:

<!DOCTYPE html>
<html lang="pt-br">
<head>
 <meta charset="UTF-8">
 <title>Página com Cabeçalhos</title>
</head>
<body>
 <h1>Título Principal</h1>
 <p>Este é um <em>texto enfatizado</em>, e isso é um <strong>texto fortemente enfatizado</strong>.</p>
 <h2>Subtítulo</h2>
 <p>Outro parágrafo sob o subtítulo.</p>
</body>
</html>

Uso de seções:

<!DOCTYPE html>
<html lang="pt-br">
<head>
 <meta charset="UTF-8">
 <title>Página com Seções</title>
</head>
<body>
 <section>
  <h1>Seção 1</h1>
  <p>Conteúdo da Seção 1.</p>
 </section>
 <section>
  <h1>Seção 2</h1>
  <p>Conteúdo da Seção 2.</p>
 </section>
</body>
</html>

3. Texto

Ênfase e forte ênfase:

<!DOCTYPE html>
<html lang="pt-br">
<head>
 <meta charset="UTF-8">
 <title>Texto com Ênfase</title>
</head>
<body>
 <p>Isso é um <em>texto enfatizado</em>, e isso é um <strong>texto fortemente enfatizado</strong>.</p>
</body>
</html>

Listas ordenadas e não ordenadas:

<!DOCTYPE html>
<html lang="pt-br">
<head>
 <meta charset="UTF-8">
 <title>Listas HTML</title>
</head>
<body>
 <ul>
  <li>Item não ordenado 1</li>
  <li>Item não ordenado 2</li>
 </ul>
 <ol>
  <li>Item ordenado 1</li>
  <li>Item ordenado 2</li>
 </ol>
</body>
</html>
